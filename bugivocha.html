About Bugivocha

Bugivocha a simple utility app for iOS that uses computer vision to extract numbers from voucher cards. Voucher cards are used to top up mobile air-time and pay for internet bundles in some countries. Instead of entering voucher numbers manually by typing each number, Bugivocha uses character recognition to extract voucher numbers from an image of a voucher thus saving time and increases accuracy.

Simple... Light-weight... Accurate

Bugivocha uses is a native application without additional dependencies which makes its bundle size extremely tiny and light-weight. Google's ML kit is a cutting edge on-device machine learning engine that can perform several tasks with outstanding accuracy and performance speed without requiring internet connection. The power of ML-kit combined with simple minimalistic and lean design makes Bugivocha easy to use and highly user friendly for all types of users.
